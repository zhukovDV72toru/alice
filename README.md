# Навык Яндекс Алисы для записи к врачу

## Запуск
1. создать .env
1. создать .venv и активировать
1. pip install -e .
1. docker compose up
1. python main.py
1. celery -A celery_app.app worker --loglevel=info --concurrency=4
1. На проде gunicorn web.server:app --config gunicorn.conf.py
